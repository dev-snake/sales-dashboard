# sql/

Catalog câu lệnh SQL cho học tập và reporting (≥ 100 queries — Phase 4).

| Thư mục | Nội dung |
|---------|----------|
| `basic/` | SELECT, WHERE, ORDER BY, LIMIT, LIKE, … |
| `intermediate/` | JOIN, GROUP BY, HAVING, subquery |
| `advanced/` | CTE, window functions |
| `reporting/` | Revenue, RFM, ABC, cohort, … |
| `optimization/` | EXPLAIN, index labs |

Hiện tại chỉ có khung thư mục. Chi tiết bài toán: `docs/sql-roadmap.md`.
